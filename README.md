# BusinessOS

BusinessOS is a lightweight, offline first business operations suite built as a single HTML file. It helps freelancers, agencies, and small teams manage clients, invoices, expenses, pricing, and contracts without relying on external tools or servers.

## Features
- Client CRM with status tracking  
- Invoice generator with print and PDF support  
- Expense splitter and settlement calculator  
- Pricing calculator using cost plus margin  
- Contract clause checklist exporter  
- Fully offline and private  
- No backend, no accounts, no dependencies  

## Usage
1. Download the HTML file  
2. Open it in any modern browser  
3. Start managing your business instantly  

All data is stored locally in your browser using localStorage.
